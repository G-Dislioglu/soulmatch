# AICOS Registry

Cross-system registry for AICOS cards, Treegraphos specs, and related reference material.

## Structure

- `cards/` — active AICOS card inventory
- `specs/` — proposal-first cross-system drafts and concept specs
- `templates/` — reusable registry templates
- `treegraphos/` — Cross-System architecture spec, hardening runs, and legacy references

## Treegraphos

Treegraphos lives here because it is not Soulmatch-only documentation. It spans the four trunks `Soulmatch`, `Maya Core`, `AICOS`, and `Bluepilot`, and extends the AICOS registry model with signals, cases, edges, and Crush runtime concepts.

- `treegraphos/TREEGRAPHOS-SPEC-v0.3.2.md`

Supporting material:
